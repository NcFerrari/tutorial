UCEL PROJEKTU:
==============
Vychozi podoba projektu, ktery slouzi k vyuce prace s rozhranimi.
V tomoto projektu jsou jeste zobrazovany sipky zavislosti
a naopak NEJSOU zobrazovany implementacni sipky.

Projekt proto slouzi pouze k demonstraci nutnosti 
potlaceni zobrazovani sipek zavislosti a naopak 
povoleni zobrazovani implementacnich sipek.


JAK PROJEKT SPUSTIT:
====================
Projekt na pocatku obsahuje pouze tridy, ktere jsou urceny k pouziti
ve vytvarenych tridach.
