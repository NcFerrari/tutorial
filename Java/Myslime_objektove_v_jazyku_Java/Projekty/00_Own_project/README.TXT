UCEL PROJEKTU:
==============
Nacvik definice jednoduchych trid vcetne jejich atributu a metod.


JAK PROJEKT SPUSTIT:
====================
Projekt je sestaven ze samostatnych trid, ktere jsou urceny k pouziti
ve vytvarenych tridach.


INSTRUKCE PRO UZIVATELE:
========================
Studenti se zde maji naucit vytvaret sve vlastni tridy pro reprezentaci 
geometrickych tvaru sestavenych z preddefinovanych primitivnich tvaru.
Pritom se nauci naprogramovat vse, s cim se seznamili pri experimentech
s projektem 02_Objekty.
